🔌 Integrações — MP + Resend
Configure as chaves de API para ativar o checkout com PIX e o envio automático de email após o pagamento.
1️⃣ Mercado Pago — Access Token

1. Acesse mercadopago.com.br/developers/panel
2. Clique em Suas integrações → Criar aplicação
3. Vá em Credenciais de produção e copie o Access Token
4. No Cloudflare: Pages → funil-clara → Settings → Environment variables
5. Adicione: MP_ACCESS_TOKEN = (seu token)

✅ Feito isso, os botões de pagamento no funil já geram PIX automaticamente.
2️⃣ Resend — Envio de email

1. Crie conta grátis em resend.com (3.000 emails/mês grátis)
2. Vá em API Keys → Create API Key e copie a chave
3. Vá em Domains → Add Domain e verifique seu domínio
4. No Cloudflare: adicione mais 2 variáveis de ambiente:
  • RESEND_API_KEY = (sua chave re_xxxxx)
  • RESEND_FROM = Clara Aureliano <receitas@seudominio.com>

✅ Depois disso, todo cliente que pagar via PIX recebe os PDFs automaticamente no email.
3️⃣ Webhook do Mercado Pago

O webhook já é configurado automaticamente a cada pagamento. Mas você pode registrar também manualmente:

1. No painel MP: Webhooks → Adicionar
2. URL: https://funil-clara.pages.dev/api/webhook
3. Eventos: marque Pagamentos

⚡ O email é enviado em segundos após o PIX ser confirmado.
